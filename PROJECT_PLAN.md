# Project Plan

## Phase 1: Data Collection
Create a dataset of Brawl Stars brawlers with stats and performance metrics.

## Phase 2: Python Analysis
Use Python and Pandas to clean the data and answer basic analysis questions.

## Phase 3: SQL Analysis
Create a database and write SQL queries to compare brawlers, roles, and game modes.

## Phase 4: Tableau Dashboard
Build visual dashboards showing win rate, pick rate, ban rate, and balance score.

## Phase 5: Balance Score
Create a custom balance score to classify brawlers as overpowered, strong, balanced, weak, or underpowered.

## Phase 6: Predictive Model
Build a model that predicts how balanced a new brawler would be based on its stats.
